# VB-909 Drum Machine
A sample-based Arduino drum machine that I built as a side-project about a year ago.

`165test` and `dfplayertest` are aptly-named test programs I built to understand what I was doing with new components I'd never used. `drum-machine` is my first attempt, which I did mostly just as a proof-of-concept. This leaves `drum_machine_2`, which is my final product — because it was to be entered into a competition soon after, it's a bit sloppy, but it works quite well.
